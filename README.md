# 图箱

一个使用选择器和正则表达式来从网站获取数据的安卓图册阅览器

## 使用说明

1.向右滑打开抽屉

2.点击添加一个新的站点

3.点击右上角的二维码扫描按钮

4.扫描站点二维码

### 如果只有手机

1.向右滑打开抽屉

2.点击添加一个新的站点

3.点击右上角的第一个按钮

4.将站点规则 JSON 内容复制粘贴进去
